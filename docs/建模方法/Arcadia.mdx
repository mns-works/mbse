# Arcadia

Arcadia 是 x 集团根据长期的工程实践，形成的一套建模方法论。
Arcadia 方法论集成于建模工具 Capella 中，通过工具内置的依托于此方法论的特有的功能，
建模人员可以方便的构建出层级清晰的系统模型。

Arcadia 将系统设计过程分为四个阶段：

- 操作分析 Operational Analysis
- 系统分析 System Analysis
- 逻辑架构 Logical Structure
- 物理架构 Physical Structure
