### 8.4.1 语义概述

SysML中构件的语义通过在KerML中定义的构件来指定，
具体通过复用Kernel语义模型库和系统模型库中的模型元素来实现。
这和KerML的Kernel层语义基于KerML的Core层来构建是类似的。
这种语义的要求通过SysML抽象表达式中包括的*语义约束*进行形式化表示。
另外，其他的语义约束要求用户模型元素之间的关系？？

特别地，有四种语义约束，用来指定SysML的语义，每一种处理一种特定种类的关系。
  1. *特化约束*。这种约束要求特定种类的`定义`或者`使用`直接或者间接的
      特化自Kernel语义库或者系统模型库中的特定的基础`Definition`或者`Usage`。
  2. *重定义约束*。
  3. *类型特征约束*。
  4. *绑定连接器约束*。
